# АРМ администратора салона красоты "Эстетика"

Веб-приложение, разработанное в рамках выпускной квалификационной работы на тему:

**«Разработка автоматизированного рабочего места (АРМ) администратора салона красоты на примере ООО Салон красоты "Эстетика"»**

## О проекте

Система предназначена для автоматизации работы администратора салона красоты.  
Приложение позволяет вести клиентскую базу, управлять мастерами и услугами, оформлять записи клиентов, проводить оплату и формировать отчёты.

Проект реализован как учебное веб-приложение с использованием Flask и SQLite.

## Основные возможности

- авторизация пользователей;
- разграничение ролей доступа;
- ведение справочника клиентов;
- ведение справочника мастеров;
- ведение справочника услуг;
- создание, редактирование и отмена записей;
- фильтрация и поиск по журналу записей;
- заметки к записям;
- проведение оплаты по записи;
- формирование квитанции;
- отчёты по выручке, мастерам, услугам и расписанию.

## Технологический стек

- Python 3
- Flask
- Flask-SQLAlchemy
- Flask-Migrate
- Flask-Login
- SQLite
- HTML
- CSS
- Bootstrap 5
