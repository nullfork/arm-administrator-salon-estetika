# АРМ администратора салона красоты "Эстетика"

Веб-приложение, разработанное в рамках ВКР на тему:

**«Разработка автоматизированного рабочего места (АРМ) администратора салона красоты на примере ООО Салон красоты "Эстетика"»**

## Назначение системы

Система предназначена для автоматизации работы администратора салона красоты:

- ведение клиентской базы;
- управление услугами и мастерами;
- запись клиентов;
- перенос и отмена записей;
- учёт оплаты;
- формирование отчётов.

## Технологический стек

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Migrate
- SQLite
- HTML / CSS / Bootstrap

## Запуск проекта

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
